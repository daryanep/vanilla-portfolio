# 🌐 Portfolio Website

A personal portfolio website built using **HTML, CSS, and JavaScript (Vanilla JS)** to showcase my profile, skills, projects, and contact information.

## 🚀 Demo

[https://daryanep.github.io/vanilla-portfolio/](https://daryanep.github.io/vanilla-portfolio/)

## ✨ Features

* About Me Section
* Skills Section
* Projects Showcase
* Contact Section

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

```bash
portfolio/
│
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## ⚙️ Getting Started

1. Clone the repository

```bash
git clone https://github.com/daryanep/vanilla-portfolio.git
```

2. Navigate to the project directory

```bash
cd vanilla-portfolio
```

3. Open `index.html` in your browser

Or use the **Live Server** extension in VS Code.
